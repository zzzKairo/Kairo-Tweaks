# Kairo Tweaks 
A multi-purpose Windows utility tool.

Includes:
- Macro creation (keyboard and mouse support, cross-application compatibility)
- Windows system tweaking and optimisation tools

## Source Code
The source code for Kairo Tweaks is currently private and is not publicly available. (Will become available in the future)

## Software Safety and Integrity
All official Kairo Tweaks releases published in the **Releases** section of the official repository are intended to be safe, legitimate system optimisation tools.
* Official releases are not designed to contain malware, ransomware, spyware, or any other harmful code.
* The software is not intended to behave as a virus, trojan, or unauthorised remote access tool.
* The application is not designed to function as a potentially unwanted program (PUA).
* No official release is intentionally created to damage, steal, or misuse user data or systems.

# Features

- Live system dashboard (CPU, RAM, GPU, disk I/O, network, process/thread/handle counts, optimisation score)
- Search Tweaks bar with risk-level filtering (Safe / Moderate / Risky)
- Quick Actions (create restore point, run all safe tweaks, clean files, set power plan, clear memory, process killer)
## Optimisation tweaks
  - Gaming & performance tweaks (scheduler, DVR, hardware scheduling, VRR, latency, Game Bar, MSI IRQ mode)
  - Power plan control, including Ultimate Performance plan activation
  - Memory management (Memory Compression, Application Pre-launch)
  - System behaviour tweaks (shutdown, power throttling, menu delay, keyboard, long paths, notifications, Store auto-download, cloud files, ink workspace, startup delay, UAC, Workplace Join, BitLocker, maintenance, OneDrive known-folder move)
  - System services optimisation/disabling
  - Network optimisation (TCP stack tuning, delivery optimisation, adapter power, DNS presets including DoH)
  - CPU & visual performance tuning (core unparking, visual effects, USB suspend)
  - Pagefile management with RAM-based profiles
  - USB & device power management
  - Windows Update control
  - WebView2-in-Search toggle
  - Svchost split threshold control
## Cleaning tools
  - File cleanup (temp files, logs, caches)
  - Component removal
  - Bloatware (UWP app) removal
  - Scheduled task disabling (telemetry, CEIP)
## Privacy tools
  - Hardware/sensor permission control
  - Security hardening
  - Privacy & telemetry tweaks (advertising ID, tracking, SIUF, CDM, background apps, search, consent store, WER, deep telemetry)
  - Customisation (Explorer/UI tweaks, dark mode, file extensions, hidden files, task view, taskbar search visibility)
  - AI & Copilot lockdown (Copilot, Recall, Click-to-Do, AI Agents, generative AI, Paint AI, Edge AI/Copilot, Office 365 Copilot, keyboard input insights, WSAIFabricSvc)
  - Smart App Control management
  - Hyper-V / Virtual Machine Platform toggle
## Gaming tools
  - Mouse & input latency optimisation
  - NVIDIA GPU deep driver tweaks with registry backup/restore
  - AMD GPU performance & telemetry tweaks
  - Intel Arc/Xe GPU tweaks
  - Audio latency reduction (DPC audio latency, exclusive mode)
  - Timer resolution control (0.5ms override)
  - DPC Latency Live Meter
  - Millisecond system response time checker
## Network diagnostics
  - Ping/latency tester (gateway and custom host)
  - Traceroute
  - Internet speed test
  - Disk health (SMART) viewer
  - System information viewer
## Extras
  - System repair tools (SFC, DISM)
  - Third-party tool launcher/installer (debloating, privacy, performance, browser categories)
- Log Viewer with filtering
- Startup Manager (enable/disable startup entries without deleting them)
- Services Viewer (view, filter, stop, or disable individual services)
- Profiles (snapshot active tweaks and reload them later)
- Per-page drift badges for reverted tweaks
- Macro Editor
  - Record, edit, and trigger keyboard + mouse macros
  - Configurable trigger, activation type, and conditional "hold to activate" trigger
  - Playback speed, repeat mode/count, and delay controls
  - Key combination (Ctrl+Alt+Enter) to hide Macro Editor nav category
- Power Plan Editor (custom power settings across multiple categories via verified powercfg GUIDs)
- Border Reveal glow effect 
- Theming system (multiple themes, including a seasonal "Christmas" particle effect option with adjustable amount/frequency/speed)
- In-app auto-updater (GitHub release checking)
- Window snap/unsnap edge-docking support
- About page with changelog, credits, and Discord/GitHub links
